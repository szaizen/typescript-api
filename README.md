## 使用法

1. 依存パッケージをインストールします。
```
$ npm i
```

2. 開発用コマンドを実行し、`http://localhost:3000` へアクセスすると、サンプルページが表示されます。
```
$ npm run dev
```

3. 本番環境へ反映する前には、本番用コマンドを実行してください。
```
$ npm run prod
```
